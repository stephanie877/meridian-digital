# Brevo Email Sequences — Copy/Paste Ready
_Import these into Brevo → Automations → Email Campaigns_

---

## HOW TO SET UP IN BREVO

1. Go to brevo.com → Create free account
2. Automations → Create Automation → "Welcome series" template
3. Trigger: "Contact added to list" (one per niche list)
4. Add 3 emails with delays: Immediately, +3 days, +7 days
5. Create 4 lists: Restoration, Electrician, GC, Auto Repair

## BREVO CONTACT FIELDS TO CREATE

When you set up Brevo, add these custom contact attributes:
- BUSINESS (text) — company name
- MOBILESCORE (number) — PageSpeed mobile score
- COPYRIGHTYEAR (number) — footer copyright year
- CITY (text) — their city
- NICHE (text) — their niche category
- WEBSITE (text) — their URL
